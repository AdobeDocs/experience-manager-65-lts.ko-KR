# 애플리케이션 및 서비스 페이지에 표시되는 항목 수 변경 {#change-the-number-of-items-displayed-on-the-applications-and-services-pages}

>[!NOTE]
> 
> 사용자에게 관리자 콘솔에 액세스할 수 있는 관리자 권한이 있는지 확인하십시오.

환경 설정 페이지에서는 각 애플리케이션 및 서비스 웹 페이지에 표시되는 서비스, 엔드포인트, 카테고리, 아카이브 수를 제어할 수 있습니다.

1. 관리 콘솔에서 **[!UICONTROL 서비스 > 애플리케이션 및 서비스 > 환경 설정]**&#x200B;을 클릭합니다.
1. 환경 설정 페이지에서 각 페이지에 표시할 수 있는 서비스, 엔드포인트, 카테고리, 아카이브 수를 입력한 후 **[!UICONTROL 저장]**&#x200B;을 클릭합니다.
