# PDF Generator 백업 제한 사항 {#pdf-generator-backup-limitations}

PDF Generator가 파일을 변환하는 데 사용하는 임시 디렉터리는 백업할 수 없습니다. 서비스가 제대로 복구되었더라도 PDF Generator가 설정된 간격으로 임시 디렉터리의 콘텐츠를 검토하고 지우기 때문에 데이터가 손실될 수 있습니다.
