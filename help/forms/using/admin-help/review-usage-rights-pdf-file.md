# PDF 파일의 사용 권한 검토 {#review-the-usage-rights-of-a-pdf-file}

Acrobat Reader DC 확장 프로그램 웹 애플리케이션에 PDF 파일을 업로드하면 해당 파일이 Reader 확장인지 여부, 적용된 사용 권한과 해당 만료 일자에 대한 정보를 확인할 수 있습니다.

1. `http://[localhost]:[port]/ReaderExtensions`(여기에서 *[port]*&#x200B;는 애플리케이션 서버의 포트 번호)에서 Acrobat Reader DC 확장 프로그램 웹 애플리케이션에 로그인합니다.
1. 찾아보기를 클릭하여 PDF 파일을 업로드한 후 정보를 클릭합니다. 파일 이름 아래에 해당 정보가 나타납니다.

   파일 사용 권한을 수정하는 방법에 대한 자세한 내용은 Acrobat Reader DC 확장 프로그램 도움말을 참조하십시오.
